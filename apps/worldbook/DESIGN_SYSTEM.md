# The World Book Web Design Contract

## 设计目标

The World Book 是 Sköpunarverk 项目群中的“世界真相 / Canon”产品。站点需要同时满足两件事：

1. 从 Sköpunarverk landing 进入时，品牌、导航和空间氛围连续；
2. 进入长文后，阅读体验更接近纸书，不让星空装饰干扰正文。

因此站点使用“两层外壳”，而不是把 Landing 页面直接复制过来。

## 两层外壳

- **Sköpunarverk 品牌层**：深色宇宙画布、轨道品牌标识、产品切换、全局页脚、青/紫交互状态。该层由 `@skopunarverk/brand` 与 `@skopunarverk/ui` 提供。
- **World Book 编辑层**：金色 Canon 信号、暖纸色页面、酒红色章节强调、衬线标题和适合中文长文的行距。该层只存在于 `apps/worldbook`。

深色背景负责“属于同一个世界系统”，暖纸色负责“这是一部可以阅读的书”。不得把正文直接放到星空背景上，也不得把共享品牌壳改成纸色。

## 颜色与可读性

- 品牌画布使用共享 `--sk-color-*`，不在 World Book 内复制一套近似色。
- 阅读纸面：`--paper #f1e8d6`、`--paper-bright #fbf6ea`。
- 正文：`--ink #2b211e`；次要正文：`--muted #6f5e56`。
- 产品强调：`--wine #842f35`；Canon/书册信号：`--gold #c89f52`。
- 低对比度星点、网格和光晕只能作为装饰，不能承载信息。
- 正文宽度不超过约 `72ch`；中文正文行高保持在 `1.75–1.9`。

## 字体层级

- 站点导航、按钮、元信息使用 sans / mono；
- 书名、章节标题、正文使用中文衬线字体栈；
- 英文全大写仅用于 kicker、卷号和系统状态，必须提高字距且保持短小；
- 标题允许高对比、大字号，正文不使用渐变字或透明字。

## 事实状态与长文组件

- Canon 使用青绿色、Derived 使用蓝色、Proposal 使用赭色、Unknown 使用紫色；每个状态必须同时显示英文状态文字，不能只靠颜色区分。
- `chapter-lede` 负责章首定位、主责专业、摘要和开放问题；`knowledge-note` 负责事实状态、正文、Canon 来源或 Derived 依赖。
- 宽表格必须放在可键盘聚焦、带可访问名称的局部横向滚动区域内；移动端不得让表格撑宽整页。
- 精确布局的 Typst 因果图作为 `worldbook-visual-island` 输出内联 SVG，并提供 `role` 与 `aria-label`；SVG 可以局部横向滚动，但不得产生整页横向溢出。
- 原生段落、表格、公式和脚注保持 HTML 语义，不得为了像素一致全部栅格化。PDF 与 HTML 共用内容和状态协议，但不要求像素级相同。

## 交互与动效

- 所有链接必须有 `hover`、`active` 和 `focus-visible` 状态；键盘焦点使用共享青色轮廓。
- 卡片悬停只做轻微位移、边框/阴影变化；不得旋转正文或制造持续运动。
- `prefers-reduced-motion: reduce` 下关闭平滑滚动和非必要过渡。
- 触控目标最小高度 44px；移动端导航允许换行，正文侧栏改为普通流。

## 页面结构

1. 全局品牌导航（共享）
2. World Book 产品 masthead 与卷册导航（产品专用）
3. 暖纸色阅读对象（首页封面、目录或 reader）
4. 授权/出版 manifest 条
5. 全局品牌页脚（共享）

共享 UI 包不得包含卷册路由、章节树、封面地图或纸面主题；这些是 World Book 的产品语义。

## 验收基线

- 1440px 与 390px 首屏都能明确看到“Sköpunarverk → World Book”的层级；
- 全局导航进入 World 时有明确 current 状态；
- 首页、四卷目录、六个公开章节、PDF、404 都使用同一壳层；
- 键盘可以依序访问跳转链接、品牌导航、产品导航和正文链接；
- 390px 下宽表格和因果图只在自身区域滚动，文档视口宽度不得超过设备宽度；
- 正文、次要正文、按钮文字与各自背景满足 WCAG AA；
- 页面在 reduced-motion 下不依赖动画表达信息。
